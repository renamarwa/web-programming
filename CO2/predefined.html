Predefined String and Math Functions in JavaScript

This page explains and demonstrates commonly used predefined functions from the String and Math objects in JavaScript.

String Object Functions

  • length – Returns the number of characters in a string.
  • toUpperCase() – Converts a string to uppercase.
  • toLowerCase() – Converts a string to lowercase.
  • charAt(index) – Returns the character at the specified index.
  • indexOf(substring) – Finds the first position of a substring.
  • substring(start, end) – Extracts characters between two indices.
  • replace(search, replaceWith) – Replaces part of a string with another value.
  • trim() – Removes whitespace from both ends of a string.
